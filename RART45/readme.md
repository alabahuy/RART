# RART45
This is 40% mechanical keyboard with Ergo layout inspired by Reviung41 by boardsource and Plaid by HGSW ,based on IMKG (Indonesia Mechanical Keyboard Group)

* Atmel328p
* Mini USB
* Ergo Layout
